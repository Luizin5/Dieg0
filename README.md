# About Diego
Dieg0 is a discord bot that use markov chains to generate text using messages send by users

# How to run
create a file named ```.env``` and put 
```
TOKEN=YOURBOTTOKEN
```
then run

```
bundle install
```
to install dependencies

and
```
discorb setup
```
to register slash commands

#### Dieg0 is under development
